### 用户注册

注册页面需要选择专业人员还是普通用户，账号不通用。

#### 注册

/user/reg

> 请求参数

```json
{
    phone:手机号
    capatcha:验证码
}
```

> 响应参数

```json
{
    code:状态码
    msg:消息
    data:
    	token:token
    	exire:过期时间
}
```

#### 获取验证码

/user/capa

> 请求参数

```json
{
    phone:手机号
}
```



> 响应参数

```json
{
    code:状态码
    msg:消息
}
```
